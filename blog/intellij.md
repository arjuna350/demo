## intellij说明
####快捷键


	运行java程序		Ctrl+Shift+R
	
	查看修改过的文件   Command + K
	逐块选择          Alt + 上箭头
	
	全局搜索          Command + Shift + F
	返回目录结构		 Alt + F1
	显示大纲          Command + F12
	打开声明          Command + B
	
	打开最近打开的文件	 Command+E
	删除一行			Command+Delete
	删除光标后行		Control+k
	
	复制本行   		command+D
